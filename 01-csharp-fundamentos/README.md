# Nivel 01 — C# Fundamentos (VS Code / Visual Studio 2022)

## Objetivo
Dominar lo básico de C# y lógica: variables, condicionales, bucles, métodos y validación.

## Reglas
- 1 práctica diaria (45–90 min)
- No avanzar sin completar checklist
- Commits pequeños por avances

## Checklist (7 días)
- [X] Día 1: Variables, tipos, entrada/salida (Console)
- [X] Día 2: if/else y validaciones
- [ ] Día 3: switch + menú
- [ ] Día 4: for / while / do-while
- [ ] Día 5: métodos (parámetros y return)
- [ ] Día 6: listas (List<T>) + búsquedas simples
- [ ] Día 7: Mini-proyecto

## Mini-proyecto (final del nivel)
CRUD en consola (en memoria) de Empleados:
- Crear
- Listar
- Buscar
- Editar
- Eliminar

**Modelo sugerido:**
Empleado: Cedula, Nombre, Departamento, PrecioHora, HorasTrabajadas
