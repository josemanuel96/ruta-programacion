# Ruta de Programación (Monorepo)

Este repositorio es mi ruta de aprendizaje por niveles.  
Cada carpeta representa un nivel o área (C#, SQL, Web, Android, DevOps).

## Cómo usar esta ruta
1. Entro al nivel que toca.
2. Leo el README del nivel.
3. Completo el checklist.
4. Hago commits pequeños por avances.

## Convención de commits (simple)
- feat: agrego algo nuevo
- fix: arreglo un bug
- docs: documentación (README, notas)
- refactor: mejoro código sin cambiar funcionalidad
